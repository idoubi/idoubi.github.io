「gvm」Go 多版本管理工具

项目地址：gvm

安装依赖

yum install -y bison

安装 gvm

bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)

source /root/.gvm/scripts/gvm

基本使用

  • 安装默认的 go 版本

yum install -y go

  • 查看所有可用版本 go

gvm listall

  • 安装指定版本 go

gvm install go1.17

  • 查看已安装版本 go

gvm list

  • 切换指定版本 go

gvm use go1.17

  • 修改环境变量
export GOPATH=/data/code/go
export GOBIN=$GOPATH/bin
export PATH=$PATH:$GOBIN
